# M7011E_Hjalmargram
A downscaled instagram used for M7011E using Python with Django for the backend and React JS for the frontend.

How to run Hjalmargram:

Backend:

>.\djangoenv\scripts\activate 

>cd hjalmargram

>py manage.py runserver


Frontend:

Open a new terminal

>cd hjalmargram-fe

>npm start

THE LAB IS DONE!!
